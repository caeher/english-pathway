## First Conditional — Primer Condicional

### Estructura
| If clause | Result clause |
|-----------|---------------|
| If + **present simple** | will + **base verb** |
| If it **rains**, | I **will stay** home. |
| If you **study** hard, | you **will pass** the exam. |

### Uso: situaciones reales y posibles en el futuro
- Cosas que **probablemente** pasarán si se cumple la condición.
- If you eat too much, you **will feel** sick.
- If she doesn't hurry, she **will be** late.

### Order: If clause first or second
| If first (comma needed) | Result first (no comma) |
|--------------------------|--------------------------|
| **If** it rains**,** I'll stay home. | I'll stay home **if** it rains. |

### Negative forms
- If you **don't study**, you **won't pass**.
- If it **doesn't rain**, we **will go** out.

### Unless = if not
- **Unless** you study, you won't pass. = **If** you **don't** study, you won't pass.

### Variations: can / may / might instead of will
- If you finish early, you **can** leave. (permission)
- If it rains, we **might** stay home. (possibility)
- If she calls, I **may** answer. (possibility)

### ⚠️ Common mistakes
- ❌ ~~If it **will rain**, I'll stay home.~~ → ✅ If it **rains**, I'll stay home.
- The **if clause** uses **present simple**, NOT future.
