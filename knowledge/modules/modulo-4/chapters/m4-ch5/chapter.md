## The Imperative — El Imperativo

### Affirmative: Base verb (sin sujeto)
- **Open** the door.
- **Sit** down, please.
- **Listen** carefully.
- **Be** quiet!

### Negative: Don't + base verb
- **Don't** open the door.
- **Don't** sit there.
- **Don't** be late!
- **Don't** worry!

### Let's + base verb (sugerencias en grupo)
- **Let's** go to the cinema.
- **Let's** eat pizza.
- **Let's not** argue.

### Uses
| Use | Example |
|-----|---------|
| **Commands** | Stop! / Be quiet! |
| **Instructions** | Turn left. Open the book. |
| **Advice/Warnings** | Be careful! / Don't touch that! |
| **Invitations** | Come in! / Have a seat! |
| **Requests (+ please)** | Close the window, please. |
| **Suggestions (let's)** | Let's go! / Let's try again! |

> 💡 El imperativo NO tiene sujeto visible. "You" está implícito.
> "Please" lo hace más cortés: **Please** sit down. / Sit down, **please**.
