## Comparatives — Comparativos

### Short adjectives: + er (+ than)
| Adjective | Comparative | Example |
|-----------|-------------|---------|
| tall | tall**er** | She is tall**er than** me. |
| big | bigg**er** | This is bigg**er than** that. |
| happy | happi**er** | I'm happi**er than** before. |
| nice | nic**er** | This is nic**er than** mine. |

### Long adjectives: more + adj (+ than)
| Adjective | Comparative | Example |
|-----------|-------------|---------|
| beautiful | **more** beautiful | She is **more beautiful than** her sister. |
| expensive | **more** expensive | This is **more expensive than** that. |
| interesting | **more** interesting | The book is **more interesting than** the movie. |

### Irregular Comparatives
| Adjective | Comparative |
|-----------|-------------|
| good | **better** |
| bad | **worse** |
| far | **farther/further** |
| much/many | **more** |
| little | **less** |

### Spelling Rules
| Rule | Example |
|------|---------|
| 1 syllable + e: + r | nice → nicer |
| 1 syllable, CVC: double + er | big → bigger, hot → hotter |
| 2 syllables ending -y: y → ier | happy → happier, easy → easier |
| 2+ syllables: more + adj | beautiful → more beautiful |

> 💡 "Than" = que (comparar): She is taller **than** me.
