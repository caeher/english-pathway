## Presentaciones en Inglés — Estructura y Frases Clave

Una buena presentación tiene una estructura clara que guía a la audiencia paso a paso.

### Las 4 Fases de una Presentación

#### 🎬 1. Opening (Apertura)
Saludo + presentación personal + tema:

| Frase | Nivel |
|---|---|
| "Good morning. My name is... Today I'm going to talk about..." | Formal |
| "Hi everyone! I'd like to start by..." | Semi-formal |
| "Let me begin with an interesting fact..." | Semi-formal |
| "Have you ever wondered why...?" | Enganche |

#### 📊 2. Body (Cuerpo)
2-3 puntos principales con transiciones:

| Frase de transición | Uso |
|---|---|
| "First of all, let's look at..." | Primer punto |
| "Moving on to my second point..." | Cambio de punto |
| "Now let's turn to..." | Cambio de tema |
| "As you can see in this chart..." | Describir visual |
| "For instance..." | Dar ejemplo |
| "I'd like to highlight that..." | Enfatizar |

#### ❓ 3. Q&A (Preguntas)
| Frase | Uso |
|---|---|
| "Does anyone have any questions?" | Abrir Q&A |
| "That's a great question." | Ganar tiempo |
| "If I understand correctly, you're asking..." | Clarificar |
| "I'll look into that and get back to you." | No saber la respuesta |

#### 🏁 4. Closing (Cierre)
| Frase | Uso |
|---|---|
| "To summarize, the key points are..." | Resumir |
| "In conclusion..." | Cerrar formalmente |
| "The main takeaway is..." | Idea principal |
| "Thank you for your attention." | Agradecer |

### De lo simple a lo complejo:
1. **Nivel 1:** Elevator pitch (30 segundos)
2. **Nivel 2:** Mini presentación (2 minutos, 2 puntos)
3. **Nivel 3:** Presentación completa (5 minutos con visual)

> 💡 **Tip:** Practica tu opening hasta que suene natural. La primera impresión es clave.
