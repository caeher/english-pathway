## It's time to talk! â€” Time to Talk!

A conversation has structure. Learn the patterns and you can adapt them to any situation.

### Structure of a basic dialog

| Turno | Function | Ejemplo |
|-------|---------|---------|
| 1 | Saludo | *Hello! How are you?* |
| 2 | Answer + question | *I'm fine, thanks! And you?* |
| 3 | Answer | *Great, thanks!* |
| 4 | Tema / Pedido | *I'd like a coffee, please.* |
| 5 | Confirmation | *Sure! Anything else?* |
| 6 | Cierre | *No, that's all. Thank you!* |

### Expressions by communicative function

**Saludar:** 
- Informal: Hello! / Hi! / Hey!
- Formal: Good morning! / Good afternoon! / Good evening!

**Presentarse:** 
- My name is... / I'm... 
- I'm from... / I come from...
- Nice to meet you! / Pleased to meet you!

**Pedir (Making requests):** 
- I'd like... / Could I have...? / Can I get...?
- May I...? / Would it be possible to...?

**Agradecer:** 
- Thank you! / Thanks! / Thanks a lot! / Thank you very much!
- I appreciate it! / That's very kind!

**Despedirse:** 
- Goodbye! / Bye! / See you! 
- See you later! / Have a nice day! / Take care!

### Helpful answers depending on the context

| Situation | Appropriate response |
|-----------|-------------------|
| Alguien te agradece | You're welcome! / No problem! / My pleasure! |
| No entiendes | Sorry, could you repeat that? / Pardon? |
| Necesitas ayuda | Excuse me, could you help me? |
| Pides disculpas | I'm sorry / Sorry about that / My apologies |
