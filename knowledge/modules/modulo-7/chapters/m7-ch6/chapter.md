## Final Review — Final Review and Comprehensive Practice

### What are you going to review?

This chapter integrates **everything** what was learned in the course:

| Area | Key Topics |
|------|-----------|
| **Tenses** | Present/Past Simple, Continuous, Perfect, Future |
| **Modals** | can, could, must, should, may, might |
| **Conditionals** | 1st, 2nd, 3rd conditional |
| **Passive Voice** | be + past participle in all tenses |
| **Reported Speech** | said/told + backshift |
| **Relative Clauses** | who, which, that, where, whose |
| **Connectors** | however, therefore, although, despite, etc. |
| **Phrasal Verbs** | look up, give up, turn on/off, etc. |
| **Idioms** | a piece of cake, under the weather, etc. |
| **Common Mistakes** | S-V agreement, articles, false friends |
